This file was originally part of the JUCE framework Copyright (c) 2017 - ROLI Ltd. from the file juce_StandaloneFilterApp.cpp

Modified 2019 by Mardigon Toler for Soundshape.

This is the JUCEApplication that runs a SoundshapeFilterWindow, which in turn contains the Soundshape_pluginAudioProcessorEditor which draws the main GUI component, MainComponent.
