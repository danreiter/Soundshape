Soundshape
VST and standalone spectral synthesizer

Soundshape Programmer's Manual

Overview

Soundshape is written as an audio plugin with the JUCE Audio Framework. The standalone application is simply the plugin running in a small wrapper program.

Important components are the Audio Processor, which interacts with the host software and is responsible for filling an audio buffer with synthesized sound in real time in response to user input, the Converter, which keeps track of the state of the keyboard and actually performs the requiredand Digital Signal Processing and synthesis, and the Main GUI Component, which coinsists of the interface and all of the components that make it up.

File input and output for storing/loading presets and importing/exporting audio is handled by the IOHandler.
